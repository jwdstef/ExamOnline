<h1 align="center">ExamOnline</h1>
# 简介

> 一个使用Java开发的在线考试网站，可以实现在线出题，在线组卷，指定班级考试等功能。

## 开发环境
1. Jetbrain IntelliJ IDEA
2. JDK1.8
3. Tomcat8.15
4. Maven 3.5
